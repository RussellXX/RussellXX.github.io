学习笔记

记录每天的学习成果

Claude Code

The hottest new programming language is English. —— Andrej Karpathy 随着 AI 技术（主要是大语言模型和 Agent 技术）的发展，“自然语言编程”逐渐进入了大众的视野。不得不承认的一点是，AI 在对可靠性和代码质量要求不高的情况下，已经能够胜任人类程序员的大多数工作，而在高要求场景下，也能够极大地提升学习和工作效率。这就不得不提到笔者在本科阶段一直坚持“古法编程”，自己上网读文档和博客，对每一行代码都字斟句酌。直到后来进组，不得不在两周写完一个项目时，才“被逼无奈“使用了 Agent（回想起来，自己当时确乎是有些”孔乙己“的影子了）。 其实笔者开始接触 AI（或者说 LLM）的时间并不晚，可以说是 2022 年 ChatGPT 发布后比较早接触到的人之一。对于这样一个技术，初次体验给人的感觉当然是震撼，但其实我并没有和有些朋友一样，觉得它的出现可以即刻解放双手，从此一切工作、任务全部交给大模型就行。原因其实也很简单：首先，大模型的能力并没有达到能够”接管一切“的地步，即便在现在来说，如何提高 AI 的可靠性依旧是计算机科学研究的重要课题，无论是后来出现 RAG 技术，还是各类 Coding Agent，都是为了提高一个本质上是概率模型的工具生成结果的可靠性；其次，笔者认为学习和思考是作为一个人应该掌握的基本能力，也就是说，尽管我们的知识储备可能永远赶不上 AI，但是去理解掌握和创造新知识的能力是 AI 学不来的，而这种能力一定是要靠对已有知识的学习和思考来获得。 因此，学会如何使用 AI 是提升工作和学习效率的必经之路，可是无论 AI 的能力再强，也不能成为我们停止学习的理由，因为对新知识的探索需要人类的智慧，而人类的智慧需要在学习中才能培养起来。至于对”AI 能否先于人类一步验证哥德巴赫猜想“的讨论，我想还不是时候。 在这篇博客中，笔者将记录自己配置并使用 Claude Code 的全过程。 注册 Anthropic 账号 准备工作 科学上网：使用质量较高的节点，最好位于美国或者英国，避免使用多人共用的廉价甚至免费节点，以防止 IP 污染。笔者注册时使用的是 Just my socks + V2Ray。 准备注册邮箱：最好使用 Gmail 或者 Outlook，不要用 QQ 或者 163 等。 接码平台：使用 Hero-SMS。 具体步骤 使用 Chrome 浏览器的无痕模式（Incognito Mode）。 打开 Claude 官网。 进入页面后，强制刷新 3 次，确保浏览器指纹与 IP 地区完全一致。 点击 Continue with Google 进行快捷登录。 后续步骤注意点： 进入 Hero-SMS 平台后，先充值，之后再左侧搜索栏查找服务项目（Claude 或 Anthropic）；国家首选美国，若提示拥堵，可切换至英国或智利等冷门地区。 账号注册成功后，尽早创建一个 API Key 或考虑升级至 Pro 版本，似乎拥有付费记录或开发者行为能够增加账号权重。 开通 Pro/Max 首先在官网的 Settings → Account 找到并复制 Account ID。 在代充平台（如 LoveYouAI）根据 Account ID 充值 Pro/Max 会员。 在 VSCode 中使用 Claude Code 插件 插件选择官方工具 Claude Code for VS Code。 ...
